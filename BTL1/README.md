# BTL1 - Classification task on various datasets

BTL1 was to ultilize pretrained models *(CNN, ViT, RNN, Transformer, ...)* on 3 kinds of datasets: text, images and multimodal (text + images).

## Datasets used

1. **Text:** [Kaggle - DBpedia Ontology: Document Classification Dataset](https://www.kaggle.com/datasets/thedevastator/dbpedia-ontology-dataset)

> This dataset features over 50K samples of articles' headers and content. Classified into one of 14 classes. Chosen for diversity and huge number of samples for both train and test set.

2. **Image:** [Kaggle - Animal Classification Dataset](https://www.kaggle.com/datasets/iamsouravbanerjee/animal-image-dataset-90-different-animals)

> This dataset features 5400 animal images spanning over 90 animal species. Chosen for the large number of classes.

3. **MultiModal (text + image):** [Arxiv - Flickr8k](https://huggingface.co/datasets/jxie/flickr8k), [Huggingface](https://huggingface.co/datasets/jxie/flickr8k)

> This dataset features 8K image - text description pair and is small enough to comfortably run directly inside Google Collab.
